# Markdown 常见的语法示例

这篇笔记用于快速查看 Markdown 的常见写法。你可以一边看示例，一边在 Obsidian 里修改和预览效果。

---

## 1. 标题

Markdown 使用 `#` 表示标题，`#` 越多，标题级别越低。

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

示例：

# 一级标题示例
## 二级标题示例
### 三级标题示例

---

## 2. 段落和换行

普通文字直接输入就是段落。

```md
这是第一段文字。

这是第二段文字。
```

如果想在同一段里强制换行，可以在行尾输入两个空格，或者直接空一行。

---

## 3. 加粗、斜体、删除线

```md
**加粗文字**
*斜体文字*
***加粗并斜体***
~~删除线~~
```

效果：

**加粗文字**  
*斜体文字*  
***加粗并斜体***  
~~删除线~~

---

## 4. 无序列表

可以使用 `-`、`*` 或 `+` 创建无序列表，推荐统一使用 `-`。

```md
- Python
- PyTorch
- 深度学习
- 计算机视觉
```

效果：

- Python
- PyTorch
- 深度学习
- 计算机视觉

## 7. 引用

引用常用于记录别人说的话、书籍原文、论文观点等。

```md
> 机器学习的核心不是记住公式，而是理解数据、模型和优化之间的关系。
```

效果：

> 机器学习的核心不是记住公式，而是理解数据、模型和优化之间的关系。

---

## 8. 行内代码

行内代码适合标记命令、变量名、函数名、文件名。

```md
使用 `conda create -n dl python=3.10` 创建环境。
```

效果：

使用 `conda create -n dl python=3.10` 创建环境。

---

## 9. 代码块

代码块适合记录完整命令或程序。

````md
```python
import torch

x = torch.tensor([1, 2, 3])
print(x)
```
````

效果：

```python
import torch

x = torch.tensor([1, 2, 3])
print(x)
```

常见代码块语言：

- `python`
- `bash`
- `powershell`
- `cpp`
- `json`
- `yaml`

## 11. 图片

图片语法比链接多一个感叹号 `!`。

```md
![图片说明](图片地址)
```

示例：

```md
![PyTorch Logo](https://pytorch.org/assets/images/pytorch-logo.png)
```

在 Obsidian 中，也可以直接粘贴图片，Obsidian 会自动生成图片链接。

## 14. Obsidian 双链

Obsidian 的特色是可以用双中括号创建笔记之间的链接。

```md
[[Python基础]]
[[PyTorch训练流程]]
[[论文阅读模板]]
```

你也可以给链接起别名：

```md
[[PyTorch训练流程|查看 PyTorch 训练流程笔记]]
```

## 16. 数学公式

Markdown 可以配合 LaTeX 写数学公式。Obsidian 支持这种写法。

行内公式：

```md
损失函数可以写成 $L = y - \hat{y}$。
```

效果：

损失函数可以写成 $L = y - \hat{y}$。

独立公式：

```md
$$
L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2
$$
```

效果：

$$
L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2
$$

```html
</details>
```

## 19. 常用快捷记法总结

| 目标 | 写法 |
|---|---|
| 标题 | `# 标题` |
| 加粗 | `**文字**` |
| 斜体 | `*文字*` |
| 删除线 | `~~文字~~` |
| 无序列表 | `- 项目` |
| 有序列表 | `1. 项目` |
| 任务列表 | `- [ ] 任务` |
| 引用 | `> 引用内容` |
| 行内代码 | `` `代码` `` |
| 代码块 | 三个反引号 |
| 链接 | `[名称](网址)` |
| 图片 | `![说明](地址)` |
| Obsidian 链接 | `[[笔记名]]` |
| 标签 | `#标签` |
| 分割线 | `---` |
