# 知识图谱
在右边的设置里面可以勾选上附件开关这样就可以更加详细的显示出知识图谱来了
# 双链、反链、图谱
1、出链：“输入[[” 即可链接。
2、反链：用于链接之前的笔记
3、有时候附件多的情况下可以在左上角新建一个文件夹，然后把单纯的笔记文件放入其中即可
4、利用工作区来进行整体加载，简化操作。
# 界面与文件管理
1、快速切换操作可以对文件进行快速的切换，左边栏第一个，
2、搜索有两个，第一个是左上角的大搜索搜索的是文件，第二个是小搜索，ctrl+f可以搜索内容
3、白板功能和思维导图的作用是一样的。多样化设计白板。适合做整体的思维导图，也可以添加附件。
4、创建今天·的笔记，类似于日记的操作，每天的日记都是当天的一个日记，不会去多开，可以记录今天的一些新的想法和一些新的idea
# 快捷键与常用功能
1、ctrl+p打开指令面板搜索功能指令
2、右上角的阅读模式与编辑模式
3、网页浏览插件可以直接在obsidian里面打开外部链接
