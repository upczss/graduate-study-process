# 十个重要语法
1、井号空格最多六个
2、斜体用一个星号包起来，*赵*
3、无序列表用-号加空格
- 赵
- 树
- 山
4、有序列表是阿拉伯数字加点号加空格
1. 赵
2. 树
3. 山
5、链接是方括号加圆括号
[网站](www.google.com)
6、引用行首加大于号加空格
> 赵树山
> 123
> 456

7、单行代码用一对单引号多行代码用三个引号

8、图片是和链接一样的只不过是前面加一个感叹号后面的网址改成图片的地址即可
9、分割线用三个-
