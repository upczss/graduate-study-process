## 实验的可重复性
通过设计相同的随机种子来保证每次实验运行的结果是相同的，方便进行验证和对比。
```python
    torch.backends.cudnn.deterministic = True

    torch.backends.cudnn.benchmark = False
```
cuDNN 是 GPU 加速卷积的底层库，它内部有几十种卷积算法，默认会随机选一种来计算，速度不同但结果近似。
设为True：强制固定只用同一种卷积算法，保证 GPU 卷积计算结果每次完全一致。
benchmark=False：关闭自动择优，配合上面deterministic=True，全程只用固定卷积算法，牺牲一点点速度，换取结果 100% 复现。
```python
torch.manual_seed(seed)#为cpu设置随机种子

    if torch.cuda.is_available():#为gpu设置随机种子

        torch.cuda.manual_seed(seed)
```

class COVID19Dataset(Dataset):

    def __init__(self, features, targets=None):
        if targets is None:
            self.targets = None
        else:
            self.targets = torch.tensor(targets, dtype=torch.float32)

        self.features = torch.tensor(features, dtype=torch.float32)

    def __getitem__(self, index):
        if self.targets is None:
            return self.features[index]
        else:
            return self.features[index], self.targets[index]

    def __len__(self):
        return len(self.features)
```

### 1. `class COVID19Dataset(Dataset)`

```
class COVID19Dataset(Dataset):
```

定义一个叫 `COVID19Dataset` 的类，并继承 PyTorch 的 `Dataset`：

```
from torch.utils.data import Dataset
```

继承之后，需要实现三个特殊方法：

- `__init__`：保存和初始化数据
- `__getitem__`：告诉 PyTorch 如何取一条数据
- `__len__`：告诉 PyTorch数据集有多大

---

### 2. `__init__`：初始化数据集

```
def __init__(self, features, targets=None):
```

创建数据集对象时会自动执行。

参数含义：

- `features`：每个样本的输入特征
- `targets`：每个样本对应的预测目标
- `targets=None`：表示目标是可选的

例如，训练集同时有输入和正确答案：

```
train_set = COVID19Dataset(train_features, train_targets)
```

测试集通常只有输入，没有正确答案：

```
test_set = COVID19Dataset(test_features)
```

下面这部分负责处理目标：

```
if targets is None:
    self.targets = None
else:
    self.targets = torch.tensor(targets, dtype=torch.float32)
```

如果没有传入 `targets`，就记录为 `None`；如果传入了，则把 NumPy 数组等数据转换成 PyTorch Tensor。

例如：

```
targets = [10, 20, 30]
```

转换后大致是：

```
tensor([10., 20., 30.])
```

`dtype=torch.float32` 表示使用 32 位浮点数，这是神经网络训练中最常用的数据类型。

同样，输入特征也会被转换：

```
self.features = torch.tensor(features, dtype=torch.float32)
```

### 3. `__getitem__`：取出一条数据

```
def __getitem__(self, index):
```

当你写：

```
dataset[0]
```

Python 实际会执行：

```
dataset.__getitem__(0)
```

如果没有目标，说明可能是测试集：

```
if self.targets is None:
    return self.features[index]
```

只返回一条输入特征：

```
x = test_set[0]
```

如果有目标，说明通常是训练集或验证集：

```
else:
    return self.features[index], self.targets[index]
```

同时返回输入和答案：

```
x, y = train_set[0]
```

其中：

- `x` 是第 0 个样本的输入特征
- `y` 是第 0 个样本对应的 COVID-19 数值

### 4. `__len__`：返回数据集大小

```
def __len__(self):
    return len(self.features)
```

当你写：

```
len(dataset)
```

Python 实际会调用：

```
dataset.__len__()
```

假设 `features` 有 2,700 行：

```
features.shape == (2700, 93)
```

那么：

```
len(dataset) == 2700
```

因为一行代表一个样本。

class My_Model(nn.Module):

    def __init__(self, input_dim):

        # 继承父类nn.Module的初始化方法，必须写

        super(My_Model, self).__init__()

        # 搭建串行网络层

        self.layers = nn.Sequential(

            # 第一层线性层：输入维度input_dim，输出16维隐藏特征

            nn.Linear(input_dim, 16),

            # ReLU激活函数，引入非线性，提升模型拟合能力

            nn.ReLU(),

            # 第二层线性层：输入16维，输出8维隐藏特征

            nn.Linear(16, 8),

            nn.ReLU(),

            # 输出层：输入8维，输出1个预测值（回归任务）

            nn.Linear(8, 1)

        )

    # 前向传播函数，数据流过网络的逻辑，必须实现

    def forward(self, x):

        # 将输入x送入搭建好的网络层，返回预测结果

        x = self.layers(x)

        x = self.layers(x)

        return x
```

```python
def trainer(train_loader, valid_loader, model, config, device):

    # 回归任务损失函数：均方误差MSE，reduction='mean'对一个batch求平均loss

    criterion = nn.MSELoss(reduction='mean')

    # 随机梯度下降优化器，带动量momentum=0.9，学习率从配置读取

    optimizer = torch.optim.SGD(model.parameters(), lr=config['learning_rate'], momentum=0.9)

    # Tensorboard可视化日志写入器，默认生成runs文件夹记录loss曲线

    writer = SummaryWriter()

  

    # 判断./models文件夹是否存在，不存在则新建，用于保存最优模型权重

    if not os.path.isdir('./models'):

        os.mkdir('./models')

  

    # 从配置读取总训练轮数

    n_epochs = config['n_epochs']

    # 记录全局最优验证loss，初始设为无穷大，后续更小值才更新

    best_loss = math.inf

    # tensorboard绘图累计步数，每训练一个batch自增

    step = 0

    # 早停计数器：连续多轮验证loss不下降则停止训练

    early_stop_count = 0

  

    # 外层循环：遍历每一轮训练

    for epoch in range(n_epochs):

        # 切换模型至训练模式（启用梯度计算）

        model.train()

        # 列表记录本epoch所有batch的loss，后续求平均训练损失

        loss_record = []

        # tqdm生成训练进度条，position=0单条进度，leave=True训练完成保留进度条

        train_pbar = tqdm(train_loader, position=0, leave=True)

  

        # 遍历训练集每个batch的数据

        for x, y in train_pbar:

            # 梯度清零，防止上一轮梯度累积

            optimizer.zero_grad()

            # 将特征、标签迁移至指定设备（CPU/GPU）

            x, y = x.to(device), y.to(device)

            # 模型前向传播，得到预测值

            pred = model(x)

            # 计算当前batch的损失（MSE）

            loss = criterion(pred, y)

            # 反向传播，自动计算各参数梯度

            loss.backward()

            # 优化器根据梯度更新模型权重

            optimizer.step()

            # Tensorboard绘图全局步数自增

            step += 1

            # detach()断开计算图，item()取出张量loss的纯数值，存入记录列表

            loss_record.append(loss.detach().item())

  

            #显示训练过程

            train_pbar.set_description(f"Epoch [{epoch + 1}/{n_epochs}]")

            train_pbar.set_postfix(loss=loss.detach().item())

        mean_train_loss = sum(loss_record) / len(loss_record)

        writer.add_scalar('Loss/Train', mean_train_loss, step)

  

        #验证过程

        model.eval()  # 切换模型至评估模式（关闭梯度计算）

        loss_record = []  # 记录验证集每个batch的loss

  

        for x, y in valid_loader:

            x, y = x.to(device), y.to(device)

            with torch.no_grad():  # 禁用梯度计算，节省显存和计算

                pred = model(x)

                loss = criterion(pred, y)# 计算当前批次的损失值

                loss_record.append(loss.detach().item())# detach()脱离计算图，item()提取张量的纯数字，存入损失记录列表

  
  

        mean_valid_loss = sum(loss_record) / len(loss_record)

        print(f"Epoch [{epoch + 1}/{n_epochs}], Train Loss: {mean_train_loss:.4f}, Valid Loss: {mean_valid_loss:.4f}")

        writer.add_scalar('Loss/Valid', mean_valid_loss, step)

  

        if mean_valid_loss < best_loss:

            best_loss = mean_valid_loss

            torch.save(model.state_dict(), config['save_path'])#保存最好的模型权重

            print(f"验证集损失下降，保存模型权重至 {config['save_path']}")

            early_stop_count = 0  # 重置早停计数器

        else:

            early_stop_count += 1

            if early_stop_count >= config['early_stop']:

                print("验证集损失连续多轮未下降，停止训练")

                return
```

```python

def shift(x, n):
    """
    沿着时间轴平移 frame 特征。

    参数：
        x: shape = [seq_len, feat_dim]
           一段语音的全部 frame 特征。
        n: 平移步数。
           n < 0：向过去移动，当前时刻取前面的 frame。
           n > 0：向未来移动，当前时刻取后面的 frame。

    边界处理：
        超出左边界时，重复第一帧；
        超出右边界时，重复最后一帧。
    """

    # 例如 n = -1：
    # 原始：[x0, x1, x2, x3]
    # 结果：[x0, x0, x1, x2]
    # 表示每个位置取“前一帧”的特征。
    if n < 0:
        # 没有更早 frame 的位置，用第一帧补齐。
        left = x[0].repeat(-n, 1)

        # x[:-1]：去掉最后一帧，使总长度保持不变。
        right = x[:n]

    # 例如 n = 1：
    # 原始：[x0, x1, x2, x3]
    # 结果：[x1, x2, x3, x3]
    # 表示每个位置取“后一帧”的特征。
    elif n > 0:
        # x[1:]：去掉前 n 帧。
        left = x[n:]

        # 没有更晚 frame 的位置，用最后一帧补齐。
        right = x[-1].repeat(n, 1)

    # n = 0 时无需移动，直接返回原始数据。
    else:
        return x

    # 沿第 0 维（时间/frame 维度）拼接，
    # 最终 shape 仍然是 [seq_len, feat_dim]。
    return torch.cat((left, right), dim=0)


def concat_feat(x, concat_n):
    """
    将当前 frame 前后相邻的多个 frame 特征拼接在一起。

    例如：
        输入 x 的 shape 为 [seq_len, 39]
        concat_n = 5

        对第 t 帧，拼接：
        [第 t-2 帧, 第 t-1 帧, 第 t 帧, 第 t+1 帧, 第 t+2 帧]

        输出 shape 为：
        [seq_len, 5 * 39] = [seq_len, 195]

    参数：
        x: shape = [seq_len, feat_dim]
        concat_n: 要拼接的连续 frame 数量，必须为奇数。

    返回：
        拼接上下文后的特征，shape = [seq_len, concat_n * feat_dim]
    """

    # concat_n 必须是奇数，才能明确地以“当前帧”为中心。
    # 合法例子：1、3、5、7
    assert concat_n % 2 == 1

    # 只使用当前 frame 时，无须拼接上下文。
    if concat_n < 2:
        return x

    # 语音中 frame 的总数量。
    seq_len = x.size(0)

    # 单一 frame 的特征数量，例如 MFCC 常见为 39。
    feat_dim = x.size(1)

    # x 原本是 [seq_len, feat_dim]。
    # repeat 后变成 [seq_len, concat_n * feat_dim]，
    # 即每一帧的特征连续复制 concat_n 次。
    x = x.repeat(1, concat_n)

    # 变形为：
    # [seq_len, concat_n, feat_dim]
    #
    # 再交换维度，变为：
    # [concat_n, seq_len, feat_dim]
    #
    # 这样 x 的第 0 维代表“相对当前帧的位置”，
    # 便于后续分别对前后 frame 进行 shift。
    x = x.view(seq_len, concat_n, feat_dim).permute(1, 0, 2)

    # 中央位置对应当前 frame。
    # 例如 concat_n = 5 时，mid = 2：
    #
    # 位置：  0       1       2       3       4
    # 含义：t-2     t-1      t      t+1     t+2
    mid = concat_n // 2

    # 为中央 frame 的前后位置填入对应的上下文 frame。
    for r_idx in range(1, mid + 1):
        # 取得未来第 r_idx 帧。
        # 例如 r_idx = 1 时，取得 t+1。
        x[mid + r_idx, :] = shift(x[mid + r_idx, :], r_idx)

        # 取得过去第 r_idx 帧。
        # 例如 r_idx = 1 时，取得 t-1。
        x[mid - r_idx, :] = shift(x[mid - r_idx, :], -r_idx)

    # 当前 shape 是 [concat_n, seq_len, feat_dim]。
    #
    # permute 后变回 [seq_len, concat_n, feat_dim]，
    # 再把 concat_n 和 feat_dim 展平，
    # 最终得到 [seq_len, concat_n * feat_dim]。
    return x.permute(1, 0, 2).reshape(seq_len, concat_n * feat_dim)
```

```python
# 数据预处理函数
def preprocess_data(
    split,
    feat_dir,
    phone_path,
    concat_nframes,
    train_ratio=0.8,
    train_val_seed=1337
):
    """
    读取并预处理语音特征和标签。

    参数：
        split: 数据划分，可为 'train'、'val' 或 'test'
        feat_dir: 存放语音特征文件的目录
        phone_path: 存放标签 txt 文件的目录
        concat_nframes: 要拼接的相邻 frame 数量
        train_ratio: 训练集占原始训练数据的比例
        train_val_seed: 划分训练集/验证集时使用的随机种子
    """

    # 共有 41 个音素类别（phoneme classes）。
    class_num = 41

    # train 与 val 共用 train_labels.txt；
    # test 没有标签，因此使用 test 模式。
    mode = "train" if split == "train" or split == "val" else "test"

    # 标签字典：
    # 键：语音文件名称（utterance id）
    # 值：该语音每个 frame 对应的音素类别编号列表
    label_dict = {}

    # 测试集没有真实标签，不需要读取标签文件。
    if mode != "test":
        # 例如：
        # split 为 train 或 val 时，读取 train_labels.txt。
        label_file_path = os.path.join(phone_path, f"{mode}_labels.txt")

        # 使用 with 可以在读取完成后自动关闭文件。
        with open(label_file_path, "r") as phone_file:
            # 逐行读取标签文件。
            for line in phone_file:
                # 去掉行尾换行符，并按空格切分。
                #
                # 例如原始一行：
                # utt_0001 12 7 7 3 20
                #
                # 切分后：
                # ['utt_0001', '12', '7', '7', '3', '20']
                line = line.strip().split()

                # line[0] 是语音文件名称；
                # line[1:] 是每一帧的标签。
                #
                # int(p) 将字符串标签（如 '12'）转换为整数（如 12）。
                label_dict[line[0]] = [int(p) for p in line[1:]]
```

# 基础全连接模块：Linear + ReLU
class BasicBlock(nn.Module):
    def __init__(self, input_dim, output_dim):
        # 初始化父类 nn.Module
        super(BasicBlock, self).__init__()

        # 线性层后接 ReLU 激活函数
        self.block = nn.Sequential(
            nn.Linear(input_dim, output_dim),
            nn.ReLU()
        )

    def forward(self, x):
        # 数据经过当前模块
        return self.block(x)


# 分类模型：输入特征 -> 多层全连接网络 -> 41 类预测结果
class Classifier(nn.Module):
    def __init__(
        self,
        input_dim,
        output_dim=41,
        hidden_layers=1,
        hidden_dim=256
    ):
        super(Classifier, self).__init__()

        self.fc = nn.Sequential(
            # 输入层：input_dim -> hidden_dim
            BasicBlock(input_dim, hidden_dim),

            # 隐藏层：hidden_dim -> hidden_dim
            *[
                BasicBlock(hidden_dim, hidden_dim)
                for _ in range(hidden_layers)
            ],

            # 输出层：输出 41 个类别的分数（logits）
            nn.Linear(hidden_dim, output_dim)
        )

    def forward(self, x):
        # 输入 x 的形状通常为：[batch_size, input_dim]
        # 输出形状为：[batch_size, 41]
        return self.fc(x)

```python
# 训练过程
# 训练函数
def trainer(train_set, val_set, train_loader, val_loader, config, model, device):
    # 多分类任务的损失函数
    criterion = nn.CrossEntropyLoss()

    # AdamW 优化器
    optimizer = torch.optim.AdamW(
        model.parameters(),
        lr=config["learning_rate"]
    )

    # 记录最佳验证集准确率
    best_acc = 0.0

    # 总训练轮数
    n_epochs = config["n_epochs"]

    # 开始训练
    for epoch in range(n_epochs):
        train_acc = 0.0
        train_loss = 0.0

        # 切换为训练模式
        model.train()

        # 遍历训练集的每个 batch
        for i, batch in enumerate(tqdm(train_loader)):
            # 取出特征和标签
            features, labels = batch

            # 将数据移动到 CPU 或 GPU
            features = features.to(device)
            labels = labels.to(device)

            # 清空上一轮的梯度
            optimizer.zero_grad()

            # 模型预测，输出每个类别的分数
            outputs = model(features)

            # 计算预测结果与真实标签之间的损失
            loss = criterion(outputs, labels)
            loss.backward()
            optimizer.step()

            _, train_pred = torch.max(outputs,dim=1)



            # 对比预测与真实标签，相等为True(1)、不等为False(0)；求和得到本batch正确样本总数，累加至总正确数
            train_acc += (train_pred.detach() == labels.detach()).sum().item()

            # 累加当前批次损失（loss是tensor，.item()转为普通数字）
            train_loss += loss.item()
```
